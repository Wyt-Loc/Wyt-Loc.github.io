对PID的理解

PID
发布日期:   2023-03-11
更新日期:   2023-03-12
文章字数:   1.3k
阅读时长:   4 分

​

对PID的解释大概就是这张图 P比例 I积分 D微分

PID控制，即Proportional (P)– Integral(I) – Derivative(D) Control, 实际上是三种反馈控制：比例控制，积分控制与微分控制的统称。

P的作用是是当前值逐渐靠近目标值，当比较接近目标时，P的控制作用就比较小了。越接近目标，P的作用越温柔。有很多内在的或者外部的因素，使控制量发生小范围的摆动。

KP：一种比例反馈控制，把输出与输入的误差e按照一定的比例K进行放大或者缩小作为系统的输入。

关于比例控制器的公式为 ： 

pwm输出增量  =  KP * (target_val - now_val);	//每次计算的输出值 = 比例系数 * （目标值 - 当前值）

这种方法简单，但是存在稳态误差 

int main(){

    float now = 0.2; //当前值
    float val = 1.0; //目标值
    float pwm;       //每次的变化值
    float Kp = 0.5;

    while(1) {
        pwm = Kp * (val - now);  //比例控制后的输出值
        now += pwm;
        now = now - 0.1;
        printf("%f",now);
        sleep(1);
    }
    return 0;
}

可以看到执行一定的次数之后变化值与误差值相等 每次的变化和误差一样，存在稳态误差，导致KP在这种情况下失去了作用。

这种情况有可能出现在控制汽车速度时，汽车的摩擦力和我们每次增加的值一样，无人机速度控制每次的输出的增量都和空气阻力大小一样。为了防止这种情况影响我们的控制。

在此引出另一个 参数 KI

I的作用是，减小静态情况下的误差，让受控物理量尽可能接近目标值。

关于KI的使用**积分控制算法 ** 该分量和误差的积分是正比关系

所以比例+积分的公式为：

pwm = KP * (target_val - now_val) +  ki∗∫误差的累计;   

//每次计算的输出值 = 比例系数 * （目标值 - 当前值） + KI * 误差累积的积分

int main(){

    float now = 0.2; //当前值
    float val = 1.0; //目标值
    float pwm;       //每次的变化值
    float Kp = 0.5;  //比例系数
    float KI = 0.05; //积分系数
    float wucha = 0; //累计误差

    while(1) {
        wucha += (val - now); //误差累积
        pwm = Kp * (val - now) + KI * (wucha);  //比例控制后的输出值
        now += pwm;
        now = now - 0.1;
        printf("%f",now);
        sleep(1);
    }
    return 0;
}

可以看到只用KP 和KI这两个参数就可以使我们这个例子比较好的停在目标位置了。但是还有一个KD。

关于KD的使用

D的作用就是让物理量的速度趋于0，只要什么时候，这个量具有了速度，D就向相反的方向用力，尽力刹住这个变化。

int main(){

    float now = 0.2; //当前值
    float val = 1.0; //目标值
    float Kp = 0.5;  //比例系数
    float KI = 0.05; //积分系数
    float KD = 0.1; //微分系数
    float wucha = 0; //累计误差
    float error_old = 0; //记录上次的误差
    float pwm;       //每次的变化值
    
    while(1) {
        wucha += (val - now); //误差累积
        pwm = Kp * (val - now) + KI * (wucha) + KD * ((val - now) - error_old);  //比例控制
        now += pwm;
        now = now - 0.1;
        error_old = (val - now); //记录上次的误差
        printf("%f",now);
        sleep(1);
    }
    return 0;
}

D的作用在这看着好像没什么作用，但是相比于没有确实是离得远的时候有一个“助力”，离得近的时候有个阻力作用。

但是理论上再说他应该是一个纯阻力才对。让人挠头的一个东西。不过影响不大。

增量型控制算法 电机速度控制

速度闭环控制就是根据单位时间获取的脉冲数（这里使用了M法测速）测量电机的速度信息，并与目标值进行比较，得到控制偏差，然后通过对偏差的比例、积分、微分进行控制，使偏差趋向于零的过程。

位置型控制算法 电机速度控制

位置闭环控制就是根据编码器的脉冲累加测量电机的位置信息，并与目标值进行比较，得到控制偏差，然后通过对偏差的比例、积分、微分进行控制，使偏差趋向于零的过程。步进电机貌似可以用这个。 正方向加脉冲数，反方向减脉冲数。但实现应该会有很多问题。如果加个检测位置的东西会简单很多。

最后总结一下就是：

比例环节:着眼于当下误差
积分环节:累计过去所有误差，积分环节将过去的所有误差累加.最后乘以一个系数��.
微分环节:预测下一时刻误差。微分环节对当前时刻的误差求导,从而”预测”下一时刻的误差.在离散环境下,其数值等于此时的误差减去上一时刻的误差.微分环节很好的解决了积分环节带来的超调问题；
