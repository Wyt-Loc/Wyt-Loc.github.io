LED汇编实现

更新日期:   2023-10-20

点亮一个LED灯：

如图：原理图如下：

找到LED电路图，搜索LED。如图：

发现是GPIO1 -> 3 引脚。 那么想要点亮它，那么必然需要GPIO1 的时钟使能，所以接下来是找到使能GPIO1的时钟寄存器。如图：

发现 27 - 26 位为控制GPIO1 时钟的寄存器，那么该如何设置才能将我们需要的GPIO1时钟使能呢？

如图：

图示为使能的说明。

我们需要的可能大部分情况下都是一直使能，现在不考虑低功耗的情况，直接赋值位11。

时钟有了，那么我们接下来就是配置IO口了，我们知道在Stm32中，点亮一个灯的步骤为：

配置时钟，配置io模式，配置io口， 最后使能，在输入高低电平即可。

IMX也是一样的，时钟有了，那么差的就是IO口的复用，和电气属性的设置，是推挽还是输入捕获等等。。。

IO口使能寄存器如图：

将 寄存器bit 3 - 0 设置为0101 也就是 0x05；这样GPIO1_3 设置为普通IO口了，

电器属性寄存器如图：

整个寄存器配置较多，慢慢看，想赚钱的学习过程必然是枯燥的，耐得住性子，才能成大事。

bit0：

为 0 是为低压摆率，为 1 是高压摆率。 压摆率就是 在示波器看波形是的跳变沿时间，如果是梯形，那么压摆率就比较低，为矩形那么压摆率相对于应该高达一点。 高速通信那么就选高压摆率。

bit1-2：

保留

bit3-5：

也就是数值越大输出能力越高。

bit7-6：

设置输出速率。

bit10-8：

保留

bit11：

开漏输出为1 关闭，为1打开。

bit12：

bit13：
bit15-14:

也是一些属性设置。

bit16：

主要就是

压摆率，速度，驱动能力，开漏，上下拉。

配置GPIO功能。设置输入输出。

这俩寄存器，配置输入输出模式，和读取电平状态。

也就是设置GPIO1_GDIR[3] = 1; GPIO1_DR[3] = 1为输出高电平， GPIO1_DR[3] = 0为输出低电平。

流程大概就是这样。

对应代码就是

// 时钟使能     CCM_CCGR1 的基地址                 Address: 20C_4000h base + 6Ch offset = 20C_406Ch
#define CCM_CCGR1_BASE (0X020C406C)

// IO复用       IOMUXC_SW_MUX_CTL_PAD_GPIO1_IO03  Address: 20E_0000h base + 68h offset = 20E_0068h
#define SW_MUX_GPIO1_IO03_BASE (0X020E0068)

// 设置电气属性 IOMUXC_SW_MUX_CTL_PAD_LCD_DATA03  Address: 20E_0000h base + 124h offset = 20E_0124h
#define SW_PAD_GPIO1_IO03_BASE (0X020E02F4)

// 控制高低电平 GPIOx_DR                          Address: 209_C000（Base address） + 0h offset
#define GPIO1_DR_BASE (0X0209C000)

// 设置输入输出模式 GPIOx_GDIR                    Address: 209_C000（Base address） + 4h offset
#define GPIO1_GDIR_BASE (0X0209C004)
